#Linked List Characteristics
1. Variable Size
2. Non - contigious memory allocation
3. Insert in O(1) TC
4. Search in  O(n) TC

#Visulaization of Linked List:
    (Node) -> data + next
    *HEAd - first Node
    *TAIL - Last Node


#Types of Linked Lists:
1. Singular - stores address of only next node.
2. Double - stores address of next node as well as previous node.
3. Circular - same as double linked list also the TAIL stores the address of the HEAD node.
